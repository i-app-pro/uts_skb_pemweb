# 🎪 Invofest Management System

Sistem manajemen event berbasis Fullstack Web yang dibuat untuk kebutuhan UTS Pemrograman Web. Aplikasi ini digunakan untuk mengelola data event, kategori kegiatan, pembicara, serta informasi dashboard secara terintegrasi menggunakan REST API dan database cloud.

---

## 📌 Fitur Utama

* 📅 Manajemen Data Event
* 🏷️ Pengelolaan Kategori
* 🎤 Data Pembicara (Speaker)
* 📊 Dashboard Informasi
* 🌐 REST API Terintegrasi
* ☁️ Database Cloud PostgreSQL
* 🚀 Deployment Frontend & Backend Online

## 🔗 API Endpoint

| Endpoint    | Method | Keterangan                 |
| ----------- | ------ | -------------------------- |
| /categories | GET    | Menampilkan kategori       |
| /events     | GET    | Menampilkan data event     |
